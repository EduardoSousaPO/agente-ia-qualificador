## 🔐 **CONTROLE DE ACESSO POR INTERFACE**

### **🚫 RESTRIÇÕES VISUAIS:**

#### **👤 MEMBER vê:**
```
❌ NÃO VÊ: Solicitações de outros usuários
❌ NÃO VÊ: Leads de outros vendedores  
❌ NÃO VÊ: Configurações do sistema
❌ NÃO VÊ: Métricas de outros membros
❌ NÃO VÊ: Dados de outras empresas
✅ VÊ: Apenas seus próprios dados
```

#### **👨‍💼 ADMIN vê:**
```
✅ VÊ: Todos os leads da empresa
✅ VÊ: Todas as conversas da empresa
✅ VÊ: Solicitações para aprovar
✅ VÊ: Performance de toda equipe
❌ NÃO VÊ: Dados de outras empresas
❌ NÃO VÊ: Configurações de billing
```

#### **👑 OWNER vê:**
```
✅ VÊ: Tudo que admin vê +
✅ VÊ: Configurações da empresa
✅ VÊ: Billing e limites
✅ VÊ: Pode promover/rebaixar usuários
❌ NÃO VÊ: Dados de outras empresas
❌ NÃO VÊ: Configurações do sistema global
```

#### **🌟 SUPER-ADMIN vê:**
```
✅ VÊ: TUDO de todas as empresas
✅ VÊ: Configurações globais do sistema
✅ VÊ: Métricas de toda a plataforma
✅ VÊ: Pode criar/editar empresas
✅ VÊ: Logs e monitoramento global
```
